# Airport 3D models for external 3D visualization

This repository provides lightweight 3D models created with Blender.
The models are intended for use in external 3D visualization and simulation environments.

## Scope
- Geometry focused on clarity and performance
- Consistent scaling and orientation
- Optimized for real-time 3D viewing

## Creation
All models are created and maintained using Blender.

## Usage
The models can be imported into compatible 3D viewers or simulation tools for visualization purposes.
